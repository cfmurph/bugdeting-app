Import transactions from CSV

Upload a CSV file. Rows are added to your manual account (same rules as the API). Requires the same API key as /budget and other endpoints when API_KEY is set on the server.

CSV format

First row must be a header with these columns (any column order; names are case-insensitive):

  • date — YYYY-MM-DD
  • description — text (can include commas if the field is quoted)
  • amount — positive number in dollars (e.g. 12.50 or 1,234.56)
  • type — income or expense

Example:

date,description,amount,type
2024-01-15,Groceries,45.67,expense
2024-01-16,"Paycheck, net",2500.00,income
